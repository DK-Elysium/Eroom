## 서비스 개요

**Eroom**은 취업 준비생의 스펙을 실제 합격자 빅데이터와 비교 분석하여  
**개인 맞춤형 취업 로드맵**, **합격 가능성 진단**, **회사 추천**, **AI 기반 스펙 코칭**을 제공하는 
**AI 취업 네비게이션 플랫폼**입니다.

“무엇을, 얼마나, 언제까지 준비해야 하는지”  
막연한 취업 준비에 명확한 기준과 방향성을 제공합니다.


## 핵심 기능 (Key Features)

###  1️. AI 기반 스펙 진단 & gap 분석
- 사용자 스펙(학력, 자격증, 인턴, 기술 스택 등)을 합격자·불합격자 데이터셋과 **비교 분석**
- 목표 직군/기업의 **합격 가능성(%)** 실시간 예측
- 합격자와의 **gap 분석 리포트** 제공

###  2. 개인 맞춤형 취업 로드맵 생성
> **“5년 뒤 배민 기획자 목표” → 현 스펙을 기반으로 가장 현실적인 경로를 자동 생성**

예시:
- 중견기업 인턴(6개월) 경험 쌓기  
- 직무 필수 자격증 취득  
- 중소기업 신입 2년  
- 포트폴리오 고도화 → 배민 기획자 이직

#### 동작 방식
- Random Forest 기반 **예상 준비 기간 예측**
- LLM 기반 **개인 맞춤형 단계별 로드맵 생성**
- 스펙·목표 난이도·경력 가능성 등을 모두 반영한 **동적 로드맵**

### 3. 합격 확률 기반 회사 추천
- 현재 내 스펙 기준으로  **합격 확률이 높은 기업/직군 추천**
- 단순 공고가 아니라 **데이터 기반 추천 엔진**

## 활용 AI 모델 및 기술

### 1. 합격률 예측 모델 (Logistic Regression)

#### 데이터 구성
- 사용자 스펙(출신 대학, 전공, 학점, 수상 경력, 자격증, 인턴 경험 등)
- 목표 기업 및 직무
- 실제 합격/불합격 여부

#### 전처리
- 숫자형 결측값 → 중앙값  
- 범주형 결측값 → 최빈값  
- 희귀 카테고리 → “Other” 카테고리로 통합
- One-Hot Encoding  
- StandardScaler 정규화  
- Train/Test = 8:2

#### 출력
- **0 ~ 1 사이의 합격 확률**  
- 현재 사용자 스펙으로 목표 기업에 합격할 확률을 출력

### 2. 취업 준비 기간 예측 모델 (Random Forest Regressor)

#### 목적
- 목표 기업 합격까지 필요한 소요 시간을  
  **개월 단위로 예측**

#### 특징
- LabelEncoder, StandardScaler 사용  
- Train/Test = 8:2  
- 평가 지표: **MAE**(Mean Absolute Error)

### 3. LLM 기반 로드맵 생성 알고리즘

- Random Forest 예측 결과(GAP, 기간)를 LLM에 전달  
- 단계별 실행 플랜 자동 생성  
- 프로젝트·자격증·인턴·기술 스택까지 포함된  
  **완전 맞춤형 로드맵 생성**

> 단순 템플릿 X  
> 매 사용자마다 완전히 새로운 로드맵 생성


## 활용 데이터

| 데이터명 | 분야 | 출처 |
|---------|------|------|
| 합격/불합격 스펙 데이터셋 | HR(채용) | 링크드인·링커리어 |
| 사용자 이력서 PDF/OCR 텍스트 | HR | 사용자 업로드 PDF |

### PDF → 데이터셋 변환 파이프라인
- `PyMuPDF` → PDF 페이지 이미지 변환  
- `EasyOCR` → 텍스트 추출  
- 정규식으로 파싱 → 필요한 스펙 항목 추출  
- CSV 포맷 데이터셋 생성


## 문제 정의

현재 취업 시장은 다음과 같은 문제를 겪고 있음:

### 1. 정보 비대칭  
- 합격에 **'실제로'** 필요한 스펙 정보와 '카더라' 정보가 혼재되어 있음

### 2. 무의미한 스펙 경쟁  
- 스펙 경쟁에 대한 불안감 때문에 불필요한 스펙까지 준비하느라 시간/비용을 낭비함

### 3. 방향성 부재  
- 목표는 있는데 **“어디서부터 어떻게”**가 불명확

### 4. 심리적 스트레스  
- “이제 뭘 해야 하지?”  
- “이 정도면 준비가 됐나?”
